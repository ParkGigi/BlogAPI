# BlogAPI

This is a simple blog API written in Python (Flask) and uses MySQL.
An admin interface is provided (in api/static/admin) written in React.

## Setup

```
$ . .env/bin/activate
$ python run.py
```

## Testing

```
$ python -m unittest
```

## Running the admin interface

```
$ cd ./api/static/admin
$ yarn
$ yarn start
```
